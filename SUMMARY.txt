🔗 API INTEGRATION GUIDE
═══════════════════════════════════════════════════════════════════════════════

For Frontend Connection:
1. Base URL: http://localhost:8000
2. Auth Header: Authorization: Bearer {jwt_token}
3. Content-Type: application/json
4. CORS: Enabled for all origins (dev mode)

Sample Frontend Code (React):
```javascript
const loginUser = async (username, password) => {
  const response = await fetch('http://localhost:8000/api/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ username, password })
  });
  const data = await response.json();
  localStorage.setItem('token', data.access_token);
  return data;
};

const getHealthData = async (patientId) => {
  const token = localStorage.getItem('token');
  const response = await fetch(
    `http://localhost:8000/api/health-data/patient/${patientId}/trends`,
    {
      headers: { 'Authorization': `Bearer ${token}` }
    }
  );
  return response.json();
};
```
